# LMS Frontend

This is the frontend for the NovaLMS project, a modern Learning Management System built with React, Vite, TypeScript, and Material UI.

## Features

- Modern, responsive UI with Material UI
- User authentication and protected routes
- Course browsing, search, and filtering
- User dashboard with progress tracking
- Instructor controls (create/manage courses, sections, lessons)
- Notifications and profile management
- Animations and transitions for a polished experience

## Getting Started

### Prerequisites

- Node.js (v18+ recommended)
- npm

### Installation

```bash
cd frontend
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Lint

```bash
npm run lint
```

## Project Structure

- `src/` — Main source code
- `components/` — Reusable UI components
- `pages/` — Route-based pages
- `auth/` — Authentication logic
- `lib/` — Utility libraries
